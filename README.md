## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/marees.git
    ```
2. Navigate to the project directory:
    ```bash
    cd marees
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

Run the application:
```bash
npm start
```

Access the app in your browser at `http://localhost:3000`.
